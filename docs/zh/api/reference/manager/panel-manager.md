# PanelManager

负责展示面板管理。

```js
import { PanelManager } from '@mapgis/web-app-framework'
```

## getInstance

- **描述**： 获取全局单例。

```js
PanelManager.getInstance()
```

## addPanel

- **描述**：添加面板。
- **参数**

  | 参数  | 描述 | 类型   | 默认值 |
  | :---- | :--- | :----- | :----- |
  | panel | 面板 | Object | -      |

## removePanel

- **描述**：移除面板。
- **参数**

  | 参数  | 描述 | 类型   | 默认值 |
  | :---- | :--- | :----- | :----- |
  | panel | 面板 | Object | -      |

## getPanels

- **描述**：获取面板集合。
