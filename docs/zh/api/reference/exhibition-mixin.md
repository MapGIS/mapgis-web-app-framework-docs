# ExhibitionMixin

## resizeExhibition

- **类型**：`Function`
- **详细**：调整展示大小，由展示面板调用

## activateExhibition

- **类型**：`Function`
- **详细**：激活展示，由展示面板调用

## deActivateExhibition

- **类型**：`Function`
- **详细**：不激活展示，由展示面板调用

## closeExhibition

- **类型**：`Function`
- **详细**：关闭展示，由展示面板调用

## onResize

- **类型**：`Function`
- **详细**：大小调整时，展示组件内部响应

## onActive

- **类型**：`Function`
- **详细**：激活时，展示组件内部响应

## onDeActive

- **类型**：`Function`
- **详细**：不激活时，展示组件内部响应

## onClose

- **类型**：`Function`
- **详细**：关闭时，展示组件内部响应
