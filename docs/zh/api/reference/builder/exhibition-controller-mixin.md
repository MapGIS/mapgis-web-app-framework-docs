# ExhibitionControllerMixin

## exhibitions

- **类型**：`Object`
- **详细**：只能 get，展示列表

## activeExhibitionId

- **类型**：`String`
- **详细**：激活的展示 Id，支持 get 和 set

## addExhibition

- **类型**：`Function`
- **详细**：添加展示

## removeExhibition

- **类型**：`Function`
- **详细**：根据展示 Id 移除展示

## openExhibitionPanel

- **类型**：`Function`
- **详细**：打开展示面板

## closeExhibitionPanel

- **类型**：`Function`
- **详细**：关闭展示面板

## switchExhibitionPanel

- **类型**：`Function`
- **详细**：切换展示面板打开和关闭
