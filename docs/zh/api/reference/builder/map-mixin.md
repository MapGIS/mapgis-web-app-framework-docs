# MapMixin

地图混入，提供 23 维地图对象和地图库对象。

## inject

### map

- **类型**：`Object`

- **详细**：二维地图对象。

### mapbox

- **类型**：`Object`

- **详细**：MapBox 库。

### viewer

- **类型**：`Object`

- **详细**：三维场景对象。

### Cesium

- **类型**：`Object`

- **详细**：Ceisum 库。

### vueCesium

- **类型**：`Object`

- **详细**：Ceisum vue 对象。
