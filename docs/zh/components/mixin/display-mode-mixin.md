# DisplayModeMixin

## computed

| 参数             | 说明               | 类型    | 可选值 | 默认值 | 版本 |
| ---------------- | ------------------ | ------- | ------ | ------ | ---- |
| mapDisplayOnly   | 是否只加载二维     | boolean |        |        |      |
| globeDisplayOnly | 是否只加载三维     | boolean |        |        |      |
| hasMapDisplay    | 是否会加载二维     | boolean |        |        |      |
| hasGlobeDisplay  | 是否会加载三维     | boolean |        |        |      |
| bothDisplay      | 是否为二三维一体化 | boolean |        |        |      |
