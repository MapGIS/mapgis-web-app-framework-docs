# MapMixin

地图混入，提供二三维地图对象和地图库对象。

## inject

| 参数      | 说明            | 类型   | 可选值 | 默认值 | 版本 |
| --------- | --------------- | ------ | ------ | ------ | ---- |
| map       | 二维地图对象    | object |        |        |      |
| mapbox    | MapBox 库       | object |        |        |      |
| viewer    | 三维场景对象    | object |        |        |      |
| Cesium    | Ceisum 库       | object |        |        |      |
| vueCesium | Ceisum vue 对象 | object |        |        |      |
