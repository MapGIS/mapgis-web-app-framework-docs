# MpAppLoader 应用加载器

加载应用的组件。

## 代码演示

```html
<template>
  <mp-app-loader :application="application" />
</template>
```

或

```html
<template>
  <mp-app-loader :application="application" design-time />
</template>
```

## API

### MpAppLoader

| 参数        | 说明         | 类型    | 默认值 |
| ----------- | ------------ | ------- | ------ |
| application | 应用对象     | Object  | -      |
| designTime  | 是否是设计时 | Boolean | false  |
