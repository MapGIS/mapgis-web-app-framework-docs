# 写在前面

MapGIS Web App Framework 是一个 WebGIS 应用的的解决方案，它基于 vue (opens new window), typescript (opens new window), ant-design-vue (opens new window)和MapGIS Client for JavaScript (opens new window)实现。

MapGIS Web App Framework 使用了最新的前端技术栈，提供了基于微件技术的应用框架和丰富的地图功能组件，它可以帮助你快速搭建 WebGIS 应用。
